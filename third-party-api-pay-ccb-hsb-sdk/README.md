# 建行惠市宝支付sdk(ccb hsb pay sdk)

## 安装

### Maven

```xml

<dependency>
    <groupId>cn.shishuihao</groupId>
    <artifactId>third-party-api-pay-ccb-hsb-sdk</artifactId>
    <version>latest</version>
</dependency>
```

### Gradle

```groovy
compile 'cn.shishuihao:third-party-api-pay-ccb-hsb-sdk:latest'
```
