# 建设银行[外联平台]支付sdk(ccb-wlpt pay sdk)

## 安装

### Maven

```xml

<dependency>
    <groupId>cn.shishuihao</groupId>
    <artifactId>third-party-api-pay-ccb-wlpt-sdk</artifactId>
    <version>latest</version>
</dependency>
```

### Gradle

```groovy
compile 'cn.shishuihao:third-party-api-pay-ccb-wlpt-sdk:latest'
```
