# 集成到springboot with JPA

## 安装

### Maven

```xml

<dependency>
    <groupId>cn.shishuihao</groupId>
    <artifactId>third-party-api-pay-spring-boot-jpa</artifactId>
    <version>latest</version>
</dependency>
```

### Gradle

``` groovy
compile 'cn.shishuihao:third-party-api-pay-spring-boot-jpa:latest'
```
